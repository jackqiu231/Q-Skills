# 本地软硬件信息收集工具

## 快速开始
直接运行即可收集当前电脑的所有软硬件信息，无需额外参数。

## 参数说明
无必填参数，返回结构化的系统信息。

## 出参说明
返回JSON格式的信息：
- `os_info`: 操作系统相关信息（名称、版本、架构）
- `cpu_info`: CPU相关信息（型号、核心数、使用率）
- `memory_info`: 内存相关信息（总容量、已用、可用、使用率）
- `disk_info`: 磁盘分区信息（各分区总容量、已用、可用、使用率）
- `installed_apps`: 已安装的软件列表（macOS下读取/Applications目录）
